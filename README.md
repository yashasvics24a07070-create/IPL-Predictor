# 🏏 IPL Winner Predictor

A **rule-based web application** that predicts the winner of an IPL match using historical match data. Users can select teams, toss winner, and toss decision to see the predicted winner and win probabilities instantly.

## 🚀 Live Demo
Deployed on Hugging Face Spaces  
👉 [Try the App](https://huggingface.co/spaces/your-username/ipl-predictor)



## 📌 Features
- Predict match winner and win probabilities
- Interactive and user-friendly interface using **Gradio**
- Rule-based logic using historical IPL match data
- Ready for deployment on Hugging Face Spaces



## 🧠 Model Details
- Rule-based probabilistic model (no ML pipeline required)
- Calculates win probabilities using historical IPL matches
- Small adjustments for toss winner and toss decision
- Modular code: `ipl_predictor.py` handles logic, `app.py` handles UI



## 🛠️ Tech Stack
- Python
- Pandas
- Gradio
- Hugging Face Spaces



## ⚙️ How It Works
1. User selects **Team 1**, **Team 2**, toss winner, and toss decision
2. Predictor calculates win probabilities based on historical matches
3. Predicted winner and probabilities are displayed instantly
